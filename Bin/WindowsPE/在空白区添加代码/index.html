# 在代码空白区添加代码 (手动)

添加的是硬编码

1、获取 MessageBox 地址，构造 ShellCode 代码

2、E8 E9 计算公式

3、在代码区手动添加代码

4、修改入口地址，指向 ShellCode
入口地址:xxxx

call 0x12345678 Call 0x77E5425F

E8 53 FE FF FF

jmp 0x2345678

E9 2B 2B 00 00

真正要跳转的地址 = E8 这条指令的下一行地址 + X

X = 真正要跳转的地址 - E8 这条指令的下一行地址

X = 00401050 - 004011FD

X = 00403b80 - 00401055

要跳转的地方 = E8 当前的地址 + 5 + X

X = 要跳转的地址 - (E8 的地址 + 5)

6A 00 6A 00 6A 00 6A 00 E8 00 00 00 00 E9 00 00 00 00

# 实战

OD 查找 MessageBox 位置。

显示模块窗口。

找到 USER32.dll, 双击进入后， Ctrl+N 打开当前模块的名称（标签）列表。在里面查找 MessageBox。
