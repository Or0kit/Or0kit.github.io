# INLINE-HOOK 改进版

上述的 INLINE HOOK 其中要跳转的裸函数内并不能定义局部变量，因为编译器并不会帮我们平衡堆栈，但我们可以在裸函数内用全局变量。这样我们写裸函数会有很多限制，但我们可以定义一个函数，在裸函数中去调用它，这样要调用的函数就没有任何限制了。
