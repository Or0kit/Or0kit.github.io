# 远程线程函数 CreateRemoteThread

CreateRemoteThread 是一个 Windows API 函数，它能够创建一个在其它进程地址空间中运行的线程。

# 函数原型

# 参数说明

  1. hProcess [in]
    线程所属进程的进程句柄.
    该句柄必须具有 PROCESS_CREATE_THREAD, PROCESS_QUERY_INFORMATION, PROCESS_VM_OPERATION, PROCESS_VM_WRITE, 和 PROCESS_VM_READ 访问权限.

  2. lpThreadAttributes [in]
    一个指向 SECURITY_ATTRIBUTES 结构的指针，该结构指定了线程的安全属性.

  3. dwStackSize [in]
    线程栈初始大小，以字节为单位，如果该值设为 0, 那么使用系统默认大小.

  4. lpStartAddress [in]
    在远程进程的地址空间中，该线程的线程函数的起始地址.

  5. lpParameter [in]
    传给线程函数的参数.

  6. dwCreationFlags [in]
    线程的创建标志

值含义
0线程创建后立即运行
CREATE_SUSPENDED 0x00000004线程创建后先将线程挂起，直到 ResumeThread 被调用.
STACK_SIZE_PARAM_IS_A_RESERVATION 0x00010000dwStackSize 参数指定为线程栈预订大小，如果 STACK_SIZE_PARAM_IS_A_RESERVATION 没有被指定，dwStackSize 参数指定为线程栈分配大小.
  1. lpThreadId [out]
    指向所创建线程 ID 的指针，如果创建失败，该参数为 NULL.

# 函数返回值

如果调用成功，返回新线程句柄.
如果失败，返回 NULL.
