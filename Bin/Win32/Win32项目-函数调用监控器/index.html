# 项目要求

1、使用内存写入的方式，实现模块隐藏.

2、IAT 表中有的 API 使用 IAT Hook 实现 不在 IAT 表的使用 Inline Hook 实现

3、将监控的 API 参数写入到文件中

4、进程间通信方式自己选择，有创新最好

5、所有的 HOOK 能够正常卸载，不能导致进程意外结束.
