# 楔子

这两天打不开博客了，上网冲浪一番才知道原因，原来是 jsDelivr 又被 DNS 污染了，详细见：https://blog.csdn.net/qq_42951560/article/details/124844571。

我用的是 shoka 主题，于是我就试着在 VSCode 中把 shoka 主题下文件中所有的 cdn.jsdelivr.net 替换为了 fastly.jsdelivr.net 。

当然文件都提前备份了一遍。

于是就解决了。

博客可以打开了。
