# get 一个 mysql 的知识

设置主键
一个表中只能有一个主键，但这一个主键可以由多个字段组成。
两种创建主键的方法：

  1. 创建表时直接在字段之后，跟 primary key 关键字。
CREATE TABLE `stu` (
   `id` tinyint unsiged primary key, 
  `sno` int(5) unsigned zerofill DEFAULT NULL,
  `sname` char(8) DEFAULT NULL,
  `ssex` char(2) DEFAULT NULL
) ENGINE=MyISAM DEFAULT CHARSET=utf8

优点：直接 缺点：只能使用一个字段作为主键
2. 创建表时直接在所有字段之后，使用 primary key (要设为主键的字段的名字 1，字段 2...)

CREATE TABLE `stu` (
   `id` tinyint unsiged, 
  `sno` int(5) unsigned zerofill DEFAULT NULL,
  `sname` char(8) DEFAULT NULL,
  `ssex` char(2) DEFAULT NULL,
  primary key(id,sno)
) ENGINE=MyISAM DEFAULT CHARSET=utf8
